Multi-Band 5G/LTE Gateway

Designed and developed complete software stack for a 5G/LTE Gateway with an inbuilt battery. Our team of experts from Connectivity, Networking and Portable Systems helped our customers ship their products ahead of time.

Developed software for 5G/LTE gateway router for mission-critical connectivity, data intensive application without compromising on security. Device is capable of operating in fixed and mobile environments. It is a compact gateway, easy to use with great performance. An integrated backup battery to ensure seamless connectivity lasting for approximately for 12 hours makes it even more favourite gateway available in the market. It can be used in creating a vehicle hotspot, temporary job site, retail locations, SOHO network or work from home etc..





Innobox built this multi-band LTE gateway on the top of Sierra wireless EM7455 M2M module. We worked in developing networking features like Port Forwarding/Filtering, VPN Passthrough, IP Passthrough, MAC Filtering, DHCP. Our team also developed. WIFI features that include dual band concurrent 2.4GHz and 5 GHz along with providing support for 802.11 b/g/n/ac. Navigation features that are supported include aGPS, NMEA protocol, local UDP and remote streaming. A web UI is developed to access gateway device. We also developed FOTA technology on this gateway for easy over the air firmware updates.

Collaborated extensively with Chipset vendor, ODMs, network operators, platform providers and clients to ensure every feature meets the global acceptable certification standards
